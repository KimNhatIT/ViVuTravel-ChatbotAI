# VivuTravel - Tour Booking Website with AI Chatbot

VivuTravel is a full-stack travel booking web application that allows users to explore and book travel tours online.  
The system integrates AI chatbot support, online payment, and real-time communication to improve user experience.

## Features

- Browse and search travel tours
- View tour details and booking information
- User authentication (JWT)
- Online payment integration with VNPay
- AI chatbot for travel assistance
- Real-time communication using Socket.io
- Image upload and management with Cloudinary

## Tech Stack

Frontend
- ReactJS
- JavaScript
- HTML / CSS

Backend
- Node.js
- ExpressJS
- MongoDB

Others
- JWT Authentication
- Cloudinary
- VNPay Payment Gateway
- Socket.io
- AI Chatbot (Groq / Gemini API)
